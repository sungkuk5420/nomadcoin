# nomadcoin
노마드 코인 만들기

### 첫 프로그래밍
```
go mod init github.com/{githubId}/{repositoryName}

이렇게 입력하면 고의 모듈이 저장될 위치를 지정하게 된다.

이후 
main.go
파일을 생성하여 
func main(){

}

함수를 만드는데 이게 기본 실행함수이다.

vscode에서 go 확장프로그램을 설치하면 필요한 모듈이 자동으로 import된다.

문자메세지 출력은 fmt.Println("It works!")

고 프로그램 실행은

go run main.go
```
